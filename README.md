taimerJS
========

Pure timer JavaScript plugin

### Usage

Include in HTML file and run
'''javascript
var t0 = Taimer(15, '', '', null, false);
'''
or 
'''javascript
var t1 = Taimer(15, 'Timer: ', ' seconds', t1', true);
'''
